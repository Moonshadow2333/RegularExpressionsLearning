### 运用正则表达式
1. 匹配n-m
```
匹配
1，1-1，1-2，1-3...
2，2-1，2-2，2-3... 
即匹配n或者n-m。（n和m都是数字）
正则表达式：
法1：/[\d]-[\d]|[\d]/gm;
法2：/[\d][-\d]+|[\d]/gm;(有bug)
法二会匹配诸如：1-3-3- 、 1-...。这些都不是我想要的。
完善法2：/[\d][-\d]+[\d]|[\d]/gm
数字开头，数字结尾。
运用：今天上午在练习下拉菜单，但是标签里面的内容忘记加上a标签了即：<li>list1
    <ul>
        <li>list1-1</li>
        <li>list1-2</li>
        <li>list1-3</li>
    </ul>
</li>
当我想把li标签里面的内容都替换成
<a href="#">list...</a>时。
正则表达式:
/(list[\d]-[\d]|list[\d])/gm;
<a href="#">$1</a>;
```
2. 条件语句
```
如果img标签包裹在a标签里面，就需要把a标签和img标签一起匹配出来。
如果img没有被a标签包裹，就只需要把img标签匹配出来。
<a href="#">  
      <img src="1.jpg">  
</a>
<img src="2.jpg">
正则表达式
/(<[Aa]\s[^>]+>\s*)?<img [^>]+>(?(1)\s*<\/[Aa]>)/gm
需要注意的地方：\s*。
因为img标签之前之后到a标签可能有若干个空格。所以需要加上\s*。
```
